# miniguia-estudos-NotebookLM

Este projeto apresenta um miniguia de estudo para um tema e consequentemente mostra uma abordagem para a criação de um, usando o NotebookLM e engenharia de prompts. O notebook criado é público e pode ser acessado por meio do link: https://notebooklm.google.com/notebook/14646ffd-57cd-41c0-b60e-4bf87ef1d748  

O NotebookLM é um assistente de pesquisa e escrita movido por IA, da Google, no qual é possível anexar fontes, sejam pdf, vídeos, áudios, sites da web ou texto copiado, para geração com elas. Isso garante mais precisão e ajuda na questão da “alucinação” presente em IAs. Ou seja, é um LLM (Large Language Model) no qual os parâmetros são as fontes que o usuário insere.  

### Tema:  

O tema escolhido é sobre um assunto de um estudo bíblico: “Como Paulo de Tarso contribui para a chegada do cristianismo até o Brasil”. O foco é mostrar como essa ferramenta pode ser de grande ajuda no aprendizado, sendo um estudo pessoal também.  

### Objetivo de estudo:  

Entender como um personagem teve importante contribuição em fatos futuros.
